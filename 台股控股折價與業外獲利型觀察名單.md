# 一、為什麼要獨立成一個觀察類別

## 楠梓電案例：教材級的財報陷阱

2025 年度自結數字：

| 項目 | 金額 |
|------|------|
| 營業收入 | 37.4 億 |
| 營業毛利 | **-0.18 億（負毛利）** |
| 營業利益 | **-4.93 億（本業虧損）** |
| 稅前淨利 | **+32.1 億** |
| 稅後淨利 | 24.27 億 |
| **EPS** | **13.38 元** |

**本業虧 4.93 億，帳面 EPS 卻 13.38 元** — 差距全來自權益法認列轉投資（滬電股份）。

## 這對你的系統的意義

用現行「品質標記」跑楠梓電：
- EPS 13.38 ✅
- PER 約 11-12 倍 ✅
- → 系統判定 **🟢 業績股**

**但這是誤判。** 系統看不出「這個 EPS 不是本業賺的」。

# 二、篩選邏輯（給 FinMind 系統用）

## 核心判斷公式

```python
def check_earnings_source(operating_income, pretax_income, revenue):
    """
    判斷獲利來源是本業還是業外
    operating_income: 營業利益
    pretax_income: 稅前淨利
    revenue: 營業收入
    """
    if operating_income is None or pretax_income is None:
        return "❓資料不足"

    # 業外損益 = 稅前淨利 - 營業利益
    non_operating = pretax_income - operating_income

    # 本業虧損但整體獲利 → 最危險
    if operating_income <= 0 and pretax_income > 0:
        return "🔺純業外獲利"

    # 業外占稅前淨利 > 50%
    if pretax_income > 0 and (non_operating / pretax_income) > 0.5:
        return "⚠️業外為主"

    # 業外占 20-50%
    if pretax_income > 0 and (non_operating / pretax_income) > 0.2:
        return "🔸業外輔助"

    return "✅本業獲利"


def calc_operating_margin(operating_income, revenue):
    """營業利益率 — 最直接的本業體質指標"""
    if not revenue or revenue == 0:
        return None
    return (operating_income / revenue) * 100
```

## FinMind 抓取欄位

`TaiwanStockFinancialStatements` 需要抓：
- `OperatingIncome`（營業利益）
- `IncomeBeforeIncomeTax`（稅前淨利）
- `Revenue`（營業收入）
- `GrossProfit`（營業毛利）← 負毛利是極端警訊

## 顯示建議

在你的表格新增兩欄：

| 欄位 | 內容 | 範例（楠梓電）|
|------|------|--------------|
| 營益率 | 營業利益 / 營收 | **-13.2%** |
| 獲利來源 | 上述判斷結果 | **🔺純業外獲利** |

# 四、判讀框架：折價是機會還是陷阱？

## 折價收斂的可能催化劑（機會）

1. **行動派基金介入** — 如 Palliser Capital 對楠梓電
2. **實際處分並回饋股東** — 楠梓電 2025 曾處分部分滬士電持股取得約 20.6 億
3. **本業轉正** — 業外獲利不再被本業虧損蓋住
4. **提高股利政策** — 把認列的獲利實際發給股東
5. **分拆或減資** — 直接釋放價值

## 折價可能是「合理的」（陷阱）

這是最容易被忽略的一面：

| 折價原因 | 說明 |
|---------|------|
| **本業持續燒錢** | 轉投資賺的被本業虧掉，價值在流失 |
| **認列 ≠ 拿到現金** | 權益法認列是帳面數字，不等於現金入袋 |
| **管理層無意變現** | 以「長期合作關係」為由不出售 |
| **資金運用效率低** | 處分後的錢放在低收益工具，而非高 ROE 再投資 |
| **股利偏低** | 資產再值錢，不分給股東就是打折 |
| **可轉債稀釋** | 在高點發債，未來有稀釋壓力 |
| **跨境/政治風險** | 台灣公司持有中國資產的估值折讓 |

## 檢查清單（進場前逐項確認）

```
□ 本業營益率是否轉正或收斂中？
□ 轉投資的認列是「持續性」還是「一次性處分」？
□ 近三年股利政策有無改善？（殖利率 vs 認列 EPS）
□ 有無行動派股東施壓？進度如何？
□ 近期有無可轉債/現增等稀釋動作？
□ 轉投資標的本身的股價是漲是跌？（NAV 會浮動）
□ 管理層對變現的公開表態？
```

# 五、與你既有系統的整合

## 這不是「第 32 族群」

**理由**：這是**財報結構分類**，不是產業分類。同一檔股票可以同時屬於：
- 產業族群（如楠梓電屬族群 8 PCB/CCL）
- 結構標籤（控股折價型）

**建議做法**：在追蹤工具加一個**獨立標籤欄位**，而非新增族群。

## 給 Claude Code 的整合指令

```
我的品質標記系統要新增第二個維度：「獲利來源」判斷。

【新增抓取】
FinMind TaiwanStockFinancialStatements 增加抓：
- OperatingIncome（營業利益）
- IncomeBeforeIncomeTax（稅前淨利）
- GrossProfit（營業毛利）

【新增判斷函式】
def check_earnings_source(operating_income, pretax_income):
    if operating_income is None or pretax_income is None:
        return "❓資料不足"
    if operating_income <= 0 and pretax_income > 0:
        return "🔺純業外獲利"
    non_op = pretax_income - operating_income
    if pretax_income > 0 and non_op / pretax_income > 0.5:
        return "⚠️業外為主"
    if pretax_income > 0 and non_op / pretax_income > 0.2:
        return "🔸業外輔助"
    return "✅本業獲利"

【新增欄位】
表格加兩欄：
- 營益率 = 營業利益 / 營收 × 100
- 獲利來源 = 上述判斷結果

【驗證案例】
跑完後檢查 2316 楠梓電，應該顯示：
- 營益率約 -13%
- 獲利來源 = 🔺純業外獲利
如果顯示 ✅本業獲利，代表抓錯欄位。

【顯示邏輯】
品質標記要「雙標籤」，例如：
🟢業績股 + ✅本業獲利  → 真紮實
🟢業績股 + 🔺純業外獲利 → 假象，需注意
```

# 六、重要提醒

1. **本清單是「結構篩選」不是「買進清單」** — 找出來只是起點，還要逐檔驗證
2. **NAV 會浮動** — 轉投資標的股價漲跌，母公司的 NAV 跟著變，不是固定數字
3. **折價可以持續很久** — 有些公司折價了十幾年還在折價，「便宜」不等於「會漲」
4. **注意 PTT/社群名人效應** — 這類股票常有意見領袖帶風向，追隨前先自己算過一遍
5. **臻鼎-KY 是最好的對照組** — 同樣結構卻沒折價，研究「為什麼它沒折價」比研究「為什麼楠梓電折價」更有價值
